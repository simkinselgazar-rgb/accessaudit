What it does

One tool combines the measurements auditors trust with the coverage only AI can reach.

Complete WCAG coverage

Every WCAG 2.x success criterion at A, AA, and AAA — including the hard ones: focus visibility, target size, flashing content, audio descriptions, and keyboard traps.

Deterministic ground truth

axe-core, contrast mathematics, target-size measurement, tab-order walks, and accessibility-tree extraction give every AI verdict hard numbers to stand on.

Multi-model AI analysis

Code analysis, screenshot inspection, and video/flash review run on the AI provider you choose — OpenRouter, Gemini, OpenAI, Anthropic, or your own local models.

Professional reports

Export complete ACR / VPAT-style conformance reports as HTML, DOCX, XLSX, or PDF — with element locations, affected user groups, and remediation guidance.

Full audit trail

Every AI prompt and response is saved to disk, untruncated. Every finding is tagged with the subsystem that produced it. You can verify every claim.

Private by design

The tool runs entirely on your hardware. Pages you audit and reports you generate never touch our servers — and with local models, nothing leaves your machine at all.

Get started in minutes

Requires Python 3.11+ and an API key from any major AI provider (or your own local model server).

Download and unzip

Grab the package below, or git clone the repository.

Run setup

python run.py --setup — installs all dependencies, the Playwright browser, and ffmpeg automatically.

Add your AI key

Copy settings.example.json to settings.json and paste your key — or use the in-app Settings page.

Audit

python run.py, open http://127.0.0.1:5050, paste a URL, and start your first review.

System requirements

  • Python 3.11 or newer — Windows, macOS, or Linux
  • An AI provider — an API key for OpenRouter, Google Gemini, OpenAI, or Anthropic; or a local OpenAI-compatible server (vLLM, Ollama, LM Studio)
  • ffmpeg — installed automatically during setup on most systems; needed for audio/video checks
  • ~2 GB free disk — for the Playwright browser and review evidence
